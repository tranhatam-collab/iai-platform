# IAI Life OS MVP (v2)

This demo now stores data in a local SQLite database file (`life_os_mvp.db`).

## Run

```bash
cd demo/life-os-mvp
npm install
npm start
```

Open:

- `http://localhost:3000`
- `http://localhost:3000/dashboard.html`

## What is persisted

- Weekly check-ins + computed metrics
- Run history for dashboard

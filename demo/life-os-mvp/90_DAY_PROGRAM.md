# 90_DAY_PROGRAM.md

## Phase 1 (Week 1–4)
- Fix sleep
- Reduce stress
- Build routine

## Phase 2 (Week 5–8)
- Deep work
- Skill building
- Value creation

## Phase 3 (Week 9–12)
- Contribution
- Mentoring
- Long-term planning

---

## WEEK TEMPLATE

1. Check-in
2. Review last week
3. Apply 1 change
4. Run flow
5. Reflect

---

## RULE

- 1 change per week
- no overload
- measurable progress

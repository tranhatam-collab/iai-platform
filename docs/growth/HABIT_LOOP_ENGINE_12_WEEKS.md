# 0. GOAL

Giữ user 12 tuần (critical window)

---

# 1. LOOP CƠ BẢN

Check-in -> Insight -> Action -> Improvement -> Repeat

---

# 2. WEEKLY LOOP

Mỗi tuần:

1. Check-in
2. Xem thay đổi
3. Nhận 3 actions
4. Apply 1 action
5. Track

---

# 3. PROGRESSION

## Week 1–4
- ổn định (sleep, stress)

## Week 5–8
- tăng hiệu suất (focus, work)

## Week 9–12
- tạo giá trị (contribution, long-term)

---

# 4. TRIGGER

- email nhắc nhẹ
- dashboard highlight
- community check-in

# 5. DOPAMINE RULE

User rời khi:
- không thấy thay đổi
- quá nhiều action
- UI khó
